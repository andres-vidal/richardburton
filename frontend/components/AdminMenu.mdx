# Admin menu

The admin hub: a card per admin tool. It's linked from the home footer so the
admin actions live on one page instead of cluttering the footer. Each card is a
link to its route with a title and a one-line description.

## Data source

Takes no props. The tools are a static `ACTIONS` list in the component — add a
new admin tool by adding an entry (`href`, `title`, `description`) and it slots
into the responsive grid. Currently:

- **Add publications** → `/admin/publications/new`
- **Backfill references** → `/admin/publications/references`
